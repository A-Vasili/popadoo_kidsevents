Pastel Website Style Guide

A practical system for turning the colourful Popadoo logo into a soft, harmonious light theme and a readable, hue-consistent dark theme.

Core visual principles

The logo is expressive, so the interface should feel joyful without becoming noisy. Use colour to organise and reassure, not to compete with content.

Soft first

Reserve the palest tints for large backgrounds. Stronger colours belong on small accents, borders, icons, and actions.

Balanced variety

Use one dominant pastel per section and one supporting accent. Avoid placing all six colours at equal visual weight.

Readable contrast

Pastels are decorative surfaces, not default text colours. Use deep plum in light mode and near-white text in dark mode.

Logo palette and dark equivalents

Keep the hue relationships constant. In dark mode, lower lightness substantially, reduce saturation slightly for large areas, and use brighter companion tints only for accents.

How to convert pastels for dark mode

Do not invert colours mechanically. Build a dark companion palette that keeps the logo’s hue order while controlling brightness and glare.

  1. Preserve hue identity.Lavender should still read as lavender and mint as mint; avoid shifting every colour toward blue or grey.
  2. Lower surface lightness.For large dark surfaces, reduce lightness to roughly 16–28% and slightly reduce saturation to prevent vibration.
  3. Create accent companions.Keep brighter 65–80% lightness versions for icons, borders, links, and small highlights.
  4. Protect text contrast.Use near-white primary text and muted lilac-grey secondary text. Never use the deep pastel surfaces as body text colours.
  5. Reduce decorative area.Dark themes feel heavier, so use fewer large colour blocks and more neutral space between accents.

Accessibility and consistency checklist

Contrast

  • Target at least 4.5:1 for normal text.
  • Target at least 3:1 for large text and UI boundaries.
  • Test hover, focus, selected, and disabled states separately.

Colour usage

  • Never communicate status through colour alone.
  • Use one dominant pastel family per content block.
  • Keep decorative gradients subtle and away from long text.

Motion and theming

  • Respect reduced-motion preferences.
  • Persist the selected theme without blocking page use.
  • Check every component in both themes before release.
